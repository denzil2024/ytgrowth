ytgrowth.io — AI-Powered YouTube Growth Platform
Full-stack SaaS built on FastAPI (backend) and React (frontend), deployed on Railway. Helps YouTube creators grow through competitor reverse-engineering, intent-driven SEO optimization, and Thumbnail IQ — a computer vision pipeline that benchmarks thumbnails against top-performing videos in the same niche.
Stack: Python, FastAPI, React, PostgreSQL, Railway, YouTube Analytics API, Paddle
